FAQ

1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents that represents the structure of a webpage as a hierarchical tree of objects. It allows programs, like JavaScript, to interact with and modify the content, structure, and style of a webpage dynamically. Each part of a webpage (like elements, attributes, and text content) is represented as an object in the DOM tree, and developers can use the DOM to change the web page without requiring a reload.

2. How do you select an element from the DOM?

You can select elements from the DOM using various JavaScript methods: getElementById(): Selects an element by its id attribute. Example: document.getElementById('myElement'); getElementsByClassName(): Selects elements with a specific class name (returns a live collection). Example: document.getElementsByClassName('myClass'); getElementsByTagName(): Selects elements by their tag name (returns a live collection). Example: document.getElementsByTagName('div'); querySelector(): Selects the first element that matches a CSS selector. Example: document.querySelector('.myClass'); querySelectorAll(): Selects all elements that match a CSS selector (returns a static NodeList). Example: document.querySelectorAll('.myClass');

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to manage events for its child elements. This works because of event bubbling, where events propagate from the child element up through its parent elements. In event delegation, instead of adding individual event listeners to each child element, you attach one listener to the parent, which captures events triggered by its children. 4. How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes and styles using JavaScript in the following ways: a. Manipulating Attributes getAttribute(): Retrieves the value of an attribute. Example: element.getAttribute('src'); setAttribute(): Sets or updates an attribute. Example: element.setAttribute('src', 'newImage.png'); removeAttribute(): Removes an attribute. Example: element.removeAttribute('alt'); classList: Adds, removes, or toggles classes. Example: element.classList.add('newClass'); element.classList.remove('oldClass'); element.classList.toggle('active'); b. Manipulating Styles Inline Styles: Directly modify styles using the style property. Example: element.style.color = 'red'; Accessing Computed Styles: Retrieve styles applied by CSS using getComputedStyle(). Example: const styles = window.getComputedStyle(element); console.log(styles.color); CSS Text: Set multiple styles at once using the cssText property. Example: element.style.cssText = "color: red; background-color: yellow; font-size: 16px;";
